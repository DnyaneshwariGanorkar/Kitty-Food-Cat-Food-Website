# Kitty-Food-Cat-Food-Website
Description: Developed a responsive and interactive website for a fictional cat food brand, "Kitty Food."

Key Contributions:

Built a structured and responsive webpage using HTML, CSS, and JavaScript.
Designed a mobile-friendly navigation menu with JavaScript for opening/closing.
Implemented CSS media queries to ensure adaptability across different screen sizes.
Created a visually appealing layout with structured typography and color schemes.
Applied CSS animations and transitions for a smooth user experience.
Optimized the site for performance and accessibility.
Technologies Used: HTML, CSS (Flexbox & Grid), JavaScript, Google Fonts, Responsive Design
